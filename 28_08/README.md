# Ordem dos Operadores em C++

- **Mais Alto:** `* / %`
- **Médio:** `+ -`
- **Baixo:** `!= ==`
- **Mais Baixo:** `&& || !`

# Funções de Arredondamento

- **ceil()** `arredonda pra cima`
- **floor()** `arredonda pra baixo`
- **trunc()** `tira as partes decimais`
- **round()** `vai pro mais proximo`
